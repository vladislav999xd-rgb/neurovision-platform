**Актуальность.** Анализ видеозаписей нейрохирургических операций является важным инструментом для обучения хирургов и повышения качества медицинской помощи. Однако ручная разметка видео требует значительных временных затрат — от 4 до 8 часов на один час записи. Автоматизация этого процесса с помощью методов компьютерного зрения позволяет существенно сократить трудозатраты и обеспечить объективный анализ хирургической техники.

**Цель работы** — разработка веб-платформы для автоматизированного анализа видеозаписей нейрохирургических операций с использованием методов искусственного интеллекта.

**Материалы и методы.** Разработан прототип платформы NeuroVision на базе технологий Node.js и JavaScript. Для анализа видео применяются свёрточные нейронные сети (YOLO, Mask R-CNN, U-Net). Реализованы алгоритмы сегментации фаз операции, детекции хирургических инструментов (биполярный пинцет, микроножницы, аспиратор, эндоскоп) и распознавания анатомических структур (твердая мозговая оболочка, мозжечок, черепно-мозговые нервы, сосуды).

**Результаты.** Созданная платформа обеспечивает: загрузку и хранение видеозаписей операций с метаданными; автоматическую сегментацию на 6 основных этапов (подготовка, краниотомия, идентификация, основной этап, гемостаз, закрытие); детекцию инструментов с точностью 89-96%; визуализацию результатов на интерактивном таймлайне; генерацию субтитров с описанием действий хирурга. Интерфейс системы позволяет осуществлять поиск по операциям, фильтрацию по типу вмешательства и хирургу, а также визуализировать детекции в реальном времени.

**Выводы.** Разработанная платформа демонстрирует возможность применения методов искусственного интеллекта для автоматизации анализа нейрохирургических видео. Автоматическая обработка сокращает время разметки в 5-10 раз. Система может использоваться в образовательных целях, для ретроспективного анализа и контроля качества хирургических вмешательств.

**Ключевые слова:** нейрохирургия, компьютерное зрение, машинное обучение, видеоанализ, детекция инструментов, сегментация изображений, медицинская информатика.
